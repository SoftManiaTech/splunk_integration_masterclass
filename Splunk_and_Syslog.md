Login as root user using below command

```bash
sudo su
```
 

enable syslog 

```bash
systemctl enable syslog-ng
```


Go to the directory mentioned below

```bash
cd /etc/syslog-ng
```

List the files to check if syslog-ng.conf is available

```bash
ls
```

Edit the syslog-ng.conf

```bash
vi syslog-ng.conf
```
